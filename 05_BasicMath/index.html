9 is my age will return 9 when you run parseInt because it starts with a number.

The string must start with a number and the number must be identifiable given the numeric system you specified in the radix parameter. Or you could also run replace(/[^0-9]+/g, '') to remove non-numeric text and return the number using parseInt, assuming there is only one number in the String...
